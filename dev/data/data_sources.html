Data Sources#

With the HydroMT-FIAT package you can easily access exposure and vulnerability data from different sources. The following data sources are currently supported:

  • National Structure Inventory (USA)

  • OpenStreetMap Building Footprints and Land Use

  • JRC Global flood depth-damage functions

  • HAZUS flood depth-damage functions
