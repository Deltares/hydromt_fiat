==========
User guide
==========

The purpose of HydroMT-FIAT is of course to build/ update a Delft-FIAT model using
global/ local raw input data. This requires knowlegde regarding the formats of the
input data, how to present this data to HydroMT-FIAT, and how to use HydroMT-FIAT
to build the desired Delft-FIAT model. The following pages will explain each of
these topics in more depth:
