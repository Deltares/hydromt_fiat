API reference¶

FIAT model class¶

Initialize¶

fiat.FiatModel(root[, mode, config_fn, …])

General and basic API for the FIAT model in hydroMT.

Build components¶

fiat.FiatModel.setup_config(**cfdict)

Update config with a dictionary

fiat.FiatModel.setup_basemaps(region, **kwargs)

Define the model region.

fiat.FiatModel.setup_hazard(map_fn[, …])

Add a hazard map to the FIAT model schematization.

fiat.FiatModel.setup_exposure_buildings([…])

Add a buildings value exposure map to the FIAT model schematization.

fiat.FiatModel.scale_exposure(scenario, year)

Scale the exposure to the forecast year, using the shared socioeconomic pathway (SSP) projections for population and GDP growth.

Attributes¶

fiat.FiatModel.region

Returns geometry of region of the model area of interest.

fiat.FiatModel.crs

Returns coordinate reference system embedded in staticmaps.

fiat.FiatModel.res

Returns coordinates of staticmaps.

fiat.FiatModel.root

Path to model folder.

fiat.FiatModel.config

Returns parsed model configuration.

fiat.FiatModel.staticmaps

xarray.Dataset representation of all static parameter maps

fiat.FiatModel.staticgeoms

geopandas.GeoDataFrame representation of all model geometries

High level methods¶

fiat.FiatModel.read()

Method to read the complete model schematization and configuration from file.

fiat.FiatModel.write()

Method to write the complete model schematization and configuration to file.

fiat.FiatModel.build(region[, res, write, opt])

Single method to setup and write a full model schematization and configuration from scratch

fiat.FiatModel.set_root([root, mode])

Initialized the model root.

General methods¶

fiat.FiatModel.setup_config(**cfdict)

Update config with a dictionary

fiat.FiatModel.get_config(*args[, fallback, …])

Get a config value at key(s).

fiat.FiatModel.set_config(*args)

Update the config dictionary at key(s) with values.

fiat.FiatModel.read_config([config_fn])

Parse config from file.

fiat.FiatModel.write_config([config_name, …])

Write config to <root/config_fn>

fiat.FiatModel.set_staticmaps(data[, name])

Add data to staticmaps.

fiat.FiatModel.read_staticmaps()

Read staticmaps at <root/?/> and parse to xarray Dataset.

fiat.FiatModel.write_staticmaps([compress])

Write staticmaps at <root/?/> in model ready format.

fiat.FiatModel.set_staticgeoms(geom, name)

Add geom to staticmaps

fiat.FiatModel.read_staticgeoms()

Read staticgeoms at <root/?/> and parse to dict of GeoPandas.

fiat.FiatModel.write_staticgeoms()

Write staticmaps at <root/?/> in model ready format.

fiat.FiatModel.set_forcing(data[, name])

Add data to forcing attribute which is a dictionary of xarray.DataArray.

fiat.FiatModel.read_forcing()

Read forcing at <root/?/> and parse to dict of xr.DataArray.

fiat.FiatModel.write_forcing()

write forcing at <root/?/> in model ready format.
